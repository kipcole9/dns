# Phase 4+ work plan (post-Phase-3, 2026-05-02)

The 8 outstanding items from the Phase 3 wrap, in execution order.
Each box gets ticked as it lands. Anything in **bold** is a checkpoint
where the full test suite (incl. dig integration) is re-run.

- [ ] `set_response` clears `ad` and `cd` on every response per RFC 6840 §5.7
- [ ] Integration test asserts `dig` does not show `ad` in response flags
